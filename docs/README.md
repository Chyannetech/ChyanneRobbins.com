# Documentation Index

This directory documents how ChyanneRobbins.com is built and maintained. It does not define why the work exists — that question is answered once, outside `docs/`, in [`body-of-work/BODY-OF-WORK.md`](../body-of-work/BODY-OF-WORK.md), the governing philosophy for everything published here. Start there before anything below.

---

- **[EXPERIENCE-PRINCIPLES.md](EXPERIENCE-PRINCIPLES.md)** — why the experience should feel the way it does. Foundational and stable across years; every implementation document below is checked against it.

- **[PUBLISHING.md](PUBLISHING.md)** — mechanics: folder structure, frontmatter, and the git workflow for actually publishing an entry.
- **[SITE-MAP.md](SITE-MAP.md)** — the site's routes and structure philosophy.
- **[HOMEPAGE.md](HOMEPAGE.md)** — the homepage's specific purpose and structure.
- **[INTERACTION-INVENTORY.md](INTERACTION-INVENTORY.md)** — an audit of interactions as actually built, checked against DESIGN-SYSTEM.md and EXPERIENCE-PRINCIPLES.md.
